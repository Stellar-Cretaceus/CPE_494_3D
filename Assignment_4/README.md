## Concept
This program is inspired by sidescroll endless rpg game with grim atmosphere. It is a beta (still Non-Endless) gameplay-wise.
## Controls
- Z - Slash

## Feature
- Samurai Girl: Controllable with dynamic run and slash
- Breakable Object: Black scary Stone can be slash  using Katana.
- Background: 2D Plane map on a quad, blending with grim 3D model of black shadowy forest.
