## Concept
This program is inspired by game genre called "Bullet Hell", specially Touhou Project. It is a short barrage of bullets that will continuously summon throughout the duration of gameplay. Neon-Space vaporwave theme is used for the art concept of this game. 

## Controls
- FORWARD/LEFT/BACKWARD/RIGHT - W/A/S/D
- UP/DOWN - Arrow UP/ Arrow Down

## Feature
- Samurai Girl: Controllable with flying animation, can be moved three-dimensional.
- Bullet Barrage: object that shoot at you. If touched, the game ends.
- Background: 2D Plane map on a quad, resemble space.   
- Vaporwave Mountain: Decoration Mountain for A E S T H E T I C S
